I want you to create a web app which is mobile friendly which uses map data, building size/height data, sun orientation, any flooding info or publicly availible topography data such as lidar surveys to help people create native plantings in their yard. Allow the user to select an address or draw an area on a map. ask questions such as what level of effort do they want to use, species they want to maximize, and any other questions which could help target specific plants. Then create a native planting plan for them including care info, best times to plant. allow them to create a plant manifest and guide them on suppliers that they can purchase these plants for. Automaticly save this data so that others can see these plans on the map. If two people plan for the same plot of land allow a person to switch which one they see. Eventually I want to allow user to buy the plants so make a backend which allows user to request a quote and input their email to recieve it. i will then email suppliers like pizzo or prarie moon and other chicagoland native plant suppliers to get a price list and add a fee for managing logistics. Research the plant suppliers and their availibility to find out where to buy them. The goal of this project is to increase the native gardens which support our ecosystem in the Chicagoland area. This app could eventually be expanded to other areas but I want this to first focus on chicagoland. Make it minimal, visually pleasing and above all functional. Have features for easily printing planting plans and other things that the user might need. use https://rangers.urbanrivers.org/plants/species for an example of how to show the plants being used, but expand it to more.

In the end
- Make an interactive site using good location data to create a native planting plan based on sun, soil wetness, topography and habitat goals
- Give users an actionable planting layout and plan to implement this in their yard. Allow the user to edit plant types, quantities and layout for their needs
- Provide tools for procuring these plants. Also allow users to do it on their own
- Favor diversity over uniformity and highlight ecological benifits this could provide.
